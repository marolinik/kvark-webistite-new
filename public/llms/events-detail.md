# KVARK Event Detail Page Pattern

Canonical pattern: https://kvark.ai/company/events/:slug

## Purpose
Represents individual event or news articles linked from the events listing.

## Main messages
- Detail pages contain title, date/context, and main article body.
- Content can include announcements, event recaps, and related company updates.
- Slugs are dynamic; always cite the exact canonical slug URL when available.
